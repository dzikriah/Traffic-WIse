# **App Name**: TrafficWise

## Core Features:

- Real-Time Traffic Simulation: Simulate traffic conditions in real time, adhering to vehicle volume rules.
- Traffic Status Updates: Continuously update traffic status based on vehicle volume (Smooth, Moderate, Heavy).
- Dynamic Toll Gate Adjustment: Maintain toll gate settings unless significant traffic shifts occur.
- Timestamp Tracking: Display traffic data with the current timestamp for accuracy.
- Traffic Volume Variation: Simulate changes in vehicle volume (+/- 5 to +/- 20 vehicles).
- Explanation Generation: Generate textual explanations for traffic changes using a generative AI tool to describe why conditions changed.
- Data Visualization: Visualize the traffic conditions with an intuitive user interface.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey trust and reliability.
- Background color: Light gray (#F0F2F5) for a clean and modern look.
- Accent color: Bright orange (#FF9800) to highlight important information.
- Body and headline font: 'Inter', sans-serif, providing a clean, neutral, objective look suitable for both headlines and body text.
- Use road-themed icons.
- Display key metrics prominently.
- Smooth transitions for changes.
