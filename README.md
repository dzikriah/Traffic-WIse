# Traffic Wise - Jakarta Traffic Watch

Real-time traffic simulation and AI-powered route analysis for Jakarta.

## Features
- **Live Monitoring**: Real-time simulation of traffic volume and speed at key Jakarta intersections.
- **AI Route Prediction**: Get multimodal travel estimates (Car, Motorcycle, Public Transport) with tactical advice.
- **Weather Awareness**: Integration with AI weather services to show how conditions affect travel.
- **Traffic AI Assistant**: A conversational chatbot to help you navigate the city.
